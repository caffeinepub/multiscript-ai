# Specification

## Summary
**Goal:** Add a subtle settings gear icon anchored near the footer and fix the vertical spacing between the View History button and the bottom divider line.

**Planned changes:**
- Add a Lucide `Settings` gear icon positioned 10px above the copyright footer, aligned to the far right of the container, with default color `rgba(255, 255, 255, 0.3)` and a purple glow (`#A855F7`) on hover
- Make the gear icon clickable to open the Developer Control Center modal/view
- Reduce the vertical gap between the View History button and the bottom white divider line by approximately 50%

**User-visible outcome:** Users see a subtle gear icon in the footer's far-right corner that glows purple on hover and opens the Developer Control Center when clicked. The area around the View History button appears tighter and more polished.
